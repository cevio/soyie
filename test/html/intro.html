`Soyie` 是一套无依赖的[MVVM](http://baike.baidu.com/view/3507915.htm)框架。与主流的MVVM框架相比，具有轻量级，语法简单，功能齐全的优点。作者[沈赟杰](http://webkits.cn)，为51信用卡管家公司量身定制的业务架构框架。 框架在一个月内经过4次代码重构，将文件大小控制在30K左右。如果经过GZIP压缩，文件将更小。是所有MVVM框架中体积最小的一个。框架处于一期功能完成阶段，后续我们将对框架进行更多功能的架构，包括组件上更新，使其更加能满足公司业务需求。达到减少代码量，提高开发效率的目的。 同时，配合框架的开发，我们开发了一些nodejs上的工作流工具，比如`envirs-cli`组件，对项目创建和服务启动，还有任务流程管理，都起到了很好都效果。 Soyie UI 由以下几个部分组成： * 文档碎片 － 文本标签表达式 * 文档碎片 － 属性标签表达式 * 文档碎片 － 循环标签表达式 * 指令表达式 * 双向绑定 `es-binding` * 行为事件 `es-click ...` * 基本属性增强 `es-src ...` * 自定义指令 Soyie DATA 由以下几个部分组成： * relation 依赖关系自动绑定 * watcher 观察者 * autoCompiler 自动编译层 * DataRouter 数据路由 Soyie 优势： * 代码量小 * 书写简单 容易上手 * 处理大数据页面 * 处理大范围数据变动 * 可自定义数据变动回调 * 数据路由监听到数组最深层 Soyie 给公司带来的价值： * 减少开发代码量，保证数据逻辑和UI间的协调处理 * 开发周期变短，维护容易，二次接手容易。 * 内部前端技术的沉淀，吸引更多NB开发者加入。 * 如果开源，提升公司在技术界的形象。
