es-binding 双向绑定指令

所谓双向绑定，就是在view层改变的时候通知data层，data层改变的时候通知view层。形成一个数据的闭环交互。您可以很容易通过双向绑定在页面的其他区域（controller作用域）下实时显示你修改后的数据。你能省去很多代码来绑定事件和触发事件后的UI改变。同时，也省去了数据改变时候的通知代码。您完全不用写任何一丁点多余代码来实现你的需求，而是通过在页面上写入规则标签表达式。

演示：
Input 表单双向绑定:
Binding Change: Hello {{text}}!
请输入您的名字。
Select 表单双向绑定:
Binding Change: Come home with {{who}}!
请选择您和谁回家的名字。
Radio 表单双向绑定:
Binding Change: I agree with {{agree}}!
Evio Tommy XYBK
请选择您同意谁的观点。
CheckBox 表单双向绑定:
Binding Change: order tickets? {{correct}}
预定？
请选择是否要预定机票。
TextArea 表单双向绑定:
Binding Change: write code
{{code}}
写入您的代码。
